PopX MERN App

A simple authentication application built using the MERN Stack.

Tech Stack

React

Tailwind CSS

Node.js

Express.js

MongoDB

JWT Authentication

Features

User Register

User Login

JWT Authentication

Protected Profile Page

Logout

Installation

Clone the repo

git clone https://github.com/akashyadav7771/popx.git

Install backend

cd backend
npm install
npm start

Install frontend

cd frontend
npm install
npm run dev
